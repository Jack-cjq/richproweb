# 主题配色方案参考

以下是几种不同风格的主题配色方案，可以根据需要选择：

## 方案一：现代科技风（深蓝+紫色渐变）
**特点**：科技感强，现代时尚，适合创新金融平台

### 主色调
- **主背景**：`#0F172A` (深蓝灰) / `#1E293B` (中蓝灰)
- **卡片背景**：`#FFFFFF` (纯白)
- **主品牌色**：`#6366F1` (靛蓝) / `#8B5CF6` (紫色)
- **强调色**：`#EC4899` (粉紫) / `#F59E0B` (琥珀)

### 配色代码
```javascript
primary: '#6366F1',        // 主品牌色 - 靛蓝
secondary: '#8B5CF6',     // 次要色 - 紫色
accent: '#EC4899',        // 强调色 - 粉紫
success: '#10B981',       // 成功/涨 - 翠绿
warning: '#F59E0B',       // 警告 - 琥珀
danger: '#EF4444',        // 错误/跌 - 红色
info: '#3B82F6',          // 信息 - 蓝色
```

## 使用建议

1. **方案一（现代科技风）**：适合创新、科技感强的平台
2. **方案二（活力商务风）**：适合成熟、专业的金融平台
3. **方案三（优雅奢华风）**：适合高端、礼品卡交易平台
4. **方案四（清新专业风）**：适合年轻化、友好的平台
5. **方案五（温暖商务风）**：适合服务型、用户友好的平台
6. **方案六（深色模式）**：适合现代、科技感强的平台

## 当前方案（金融级冷静 · 银灰绿蓝风）

如果需要保持当前风格但增加一些色彩，可以考虑：
- 在按钮和强调元素上使用更鲜艳的蓝色（如 `#3B82F6`）
- 在成功/积极元素上使用更鲜艳的绿色（如 `#10B981`）
- 添加一些渐变效果增加视觉层次
